# 🚀 Quick Start Guide - Health Guardian

## Get Started in 5 Minutes

### Step 1: Open the App
1. Open `index.html` in your web browser
2. You'll see the landing page with a **"Launch App"** button
3. Click the button to start

### Step 2: Complete Registration (First Time Only)
The app will guide you through a 3-step registration:

**Step 1: Personal Info**
- Enter your **Full Name** (e.g., "John Smith")
- Enter your **Age** (e.g., 65)
- Click **Continue**

**Step 2: Preferences**
- Select your **Preferred Language**
- Choose your **Gender** (optional)
- Click **Continue**

**Step 3: Voice Settings**
- Choose your **Voice Preference** (Male/Female/Neutral)
- Select **Volume Level** (Normal/High for hearing difficulties/Low for quiet reminders)
- Click **Complete Setup**

✅ **You're registered!** The app will automatically open the main dashboard.

### Step 3: Add Your First Medicine
1. Click the **"+" button** (bottom right)
2. **Upload Photo** (optional): Click camera icon to choose a medicine image
3. Fill in the details:
   - **Name**: e.g., "Aspirin"
   - **Dosage**: 500 mg
   - **Instruction**: "After meal"
   - **Time**: 09:00 AM
4. Set **Start Date** (today's date or future date)
5. Leave **End Date** empty if ongoing
6. Select **Frequency**:
   - **Daily**: Every day
   - **Weekly**: Click specific days (Mon, Tue, etc.)
   - **Monthly**: Same date each month
   - **Custom Cycle**: Enter intake days (1) and pause days (2) for patterns
7. Add **Notes** if needed (e.g., "Take with water")
8. Click **"Add Medicine"** ✅

### Step 4: View Today's Medicines
1. Go to **Home** tab (bottom navigation)
2. You'll see all medicines scheduled for today
3. Each medicine shows:
   - 💊 Medicine image
   - Name and dosage
   - ⏰ Time to take
   - 📍 Intake instruction
   - Status circle: Click to mark as taken/missed/skipped

### Step 5: Handle Reminders
At the scheduled time:
1. **Full-screen alert** appears with medicine details
2. **Voice reminder** plays in your language
3. Phone **vibrates** (if enabled)
4. Choose one of three actions:
   - ✓ **Done**: Mark medicine as taken
   - ⏰ **Remind Later**: Snooze for 15 minutes
   - ✕ **Skip**: Mark medicine as skipped

### Step 6: Track Your Progress
1. Go to **Progress** tab
2. Select a medicine from the dropdown
3. View the calendar:
   - 🟢 **Green**: Medicine taken
   - 🔴 **Red**: Medicine missed
   - 🟡 **Yellow**: Medicine skipped
4. Navigate months to review history

## Common Tasks

### Edit an Existing Medicine
1. Go to **Medicines** tab
2. Find your medicine in the list
3. Click the **✏️ edit** button
4. Update any information
5. Click **"Update Medicine"**

### Delete a Medicine
1. Go to **Medicines** tab
2. Find your medicine
3. Click the **🗑️ delete** button
4. Confirm deletion

### Change Language or Voice
1. Go to **Settings** tab
2. Under "Language & Voice":
   - Select new **Language**
   - Choose new **Voice** (Male/Female/Neutral)
   - Select **Volume Level**
3. Changes apply immediately

### Enable Large Text (for better visibility)
1. Go to **Settings** tab
2. Under "Accessibility":
   - Check **"Larger Text Size"**
   - Check **"High Contrast Mode"** (for better contrast)
3. Text size increases immediately

### Export Your Data (Backup)
1. Go to **Settings** tab
2. Click **"Export Data"** button
3. A JSON file downloads with all your medicines and progress
4. Save this file safely for backup

### Clear All Data (Start Fresh)
1. Go to **Settings** tab
2. Click **"Clear All Data"** button
3. Confirm twice (this cannot be undone!)
4. App resets and registration page appears

## Tips & Tricks

### 📸 Medicine Photos
- Upload a clear photo of your medicine
- Takes up very little storage space
- Helps you identify the right medicine quickly
- Falls back to placeholder if not available

### 🔔 Setting Reminders
- Set reminder time for when you'll definitely be awake
- Voice reminder repeats every 3 seconds until acknowledged
- You can snooze for 15 minutes if not ready yet
- Vibration + Voice + Full screen = Can't miss it!

### 📅 Tracking Adherence
- Check progress daily to stay motivated
- Green indicators show good adherence
- Red/yellow shows which doses were missed
- Review monthly progress to identify patterns

### 🌍 Language Support
- 11 languages available
- Voice changes to match selected language
- Important for non-English speakers
- Makes reminders more personal and understandable

### ♿ Accessibility Options
- **Large Text**: Increases all font sizes (helpful for weak eyesight)
- **High Contrast**: Better contrast for visibility
- **Vibration**: Physical notification (helpful if can't hear)
- **Volume Levels**: High volume for hard of hearing users

### 💾 Data Safety
- All data stored locally in browser
- Export your data regularly as backup
- No data is sent to any server
- Completely private and offline

## Troubleshooting

### Q: Reminders not appearing?
A: 
- Check browser notification permissions
- Make sure browser tab is open (some browsers pause inactive tabs)
- Check system time is correct
- Try refreshing the page

### Q: Voice reminder not playing?
A:
- Check device volume is not muted
- Verify browser has microphone/speaker access
- Some browsers need internet for voice synthesis
- Try a different language

### Q: Data disappeared?
A:
- Clearing browser cache/cookies removes stored data
- Try using a different browser to check
- Export regularly to avoid data loss
- Private/Incognito mode doesn't save data

### Q: Can I use on multiple devices?
A:
- Each device stores data separately
- You need to export data from one device and manually add medicines on another
- No cloud sync in this version (future enhancement)
- Each device has independent data

### Q: Is my data private?
A:
- ✅ Yes! All data stored locally in your browser
- ❌ No data sent to any server
- ❌ No analytics or tracking
- ❌ No internet required (except for voice synthesis)
