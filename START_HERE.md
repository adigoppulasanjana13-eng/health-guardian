## ❓ Frequently Asked Questions

**Q: Do I need to install anything?**  
A: No! Just open index.html in browser.

**Q: Is my data safe?**  
A: Yes! All data stays on your device, never uploaded.

**Q: Can I use on multiple devices?**  
A: Each device has separate data (sync in v2.0).

**Q: Does it work offline?**  
A: Yes, completely offline except voice synthesis.

**Q: What if I lose my data?**
