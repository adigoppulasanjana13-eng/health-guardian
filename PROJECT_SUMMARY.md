## 🚀 Getting Started

### Step 1: Open the App
```
1. Open "index.html" in your web browser
2. Click "Launch App"
3. Complete 3-step registration
```

### Step 2: Add Medicines
```
1. Click "+" button
2. Fill in medicine details
3. Select frequency type
4. Click "Add Medicine"
```

### Step 3: Set Reminders
```
1. At scheduled time, app shows alert
2. Click "Done" to mark taken
3. Or "Remind Later" or "Skip"
```

### Step 4: Track Progress
```
1. Go to Progress tab
2. Select a medicine
3. View calendar with color coding
4. Check your adherence history
```
